# find-jobs

Learning Vue JS for the very first time.

## Recommended IDE Setup

## Project Clone

```sh
git clone https://github.com/Johndiddles/find-jobs
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
